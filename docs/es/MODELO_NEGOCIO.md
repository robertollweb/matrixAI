# Modelo de negocio de MatrixAI

El core y la CLI de MatrixAI son software libre publicados bajo la GNU Affero General Public License version 3 (AGPL v3). El registro de verificacion de licencia esta disponible en [VERIFICACION_LICENCIA.md](VERIFICACION_LICENCIA.md). El nucleo publico debe seguir siendo usable, auditable y redistribuible por la comunidad.

## Que es gratuito

- Lenguaje, runtime, CLI, entrenamiento, registry, servidor HTTP, herramientas de despliegue y documentacion.
- Codigo fuente publico bajo AGPL v3.
- Uso, modificacion, redistribucion y contribucion comunitaria bajo los terminos de la licencia.

## Capa comercial futura

MatrixAI Studio como producto descargable pulido queda previsto despues del roadmap v1.0, solo si la adopcion real demuestra demanda. Ese Studio futuro deberia comunicarse con el core AGPL por HTTP en lugar de importar modulos Python directamente.

La arquitectura esta disenada para mantener separados el core AGPL y cualquier Studio comercial futuro. Las implicaciones legales y comerciales de esa separacion deben validarse con un profesional cualificado antes de lanzar un producto de pago.

## Donativos

Se podran aceptar donativos voluntarios mediante GitHub Sponsors o una plataforma equivalente para ayudar a cubrir costes de servidor y tiempo de desarrollo. Los donativos no son necesarios para usar el core.

## Aviso fiscal y legal

Este documento es un registro de decision de producto, no asesoramiento legal, fiscal ni contable. La gestion de donativos, ingresos comerciales, IVA/impuestos y cualquier licencia futura de pago deben revisarse con profesionales cualificados antes del lanzamiento.

## Marca

MatrixAI, matrixaistudio.org y la identidad publica del proyecto pertenecen al autor del proyecto. Los proveedores externos solo se mencionan cuando sea tecnicamente necesario para describir compatibilidad configurable.
